# 6. Protein calculations

Create a pure Kotlin protein calculation function separate from Compose UI.

Use the same proportional logic as ingredient calories.

Formula:

```text
protein used =
protein per reference quantity
× quantity used
÷ reference quantity
```

Example:

```text
22 g protein per 100 g
300 g used

22 × 300 ÷ 100 = 66 g protein
```

For count-based ingredients:

```text
protein used =
protein per unit × number of units
```

## Missing values

If an ingredient has no assigned protein:

* Its protein amount is unknown.
* Do not automatically count it as 0 g.
* Mark the containing recipe as having incomplete protein data.

A recipe should expose:

* Known protein total
* Whether protein data is complete
* Number of ingredients missing protein

Example:

```text

```

If all ingredients have assigned protein:

```text

```
